# Alzheimer's Disease Detection using Deep Learning

## Overview
This project uses Deep Learning techniques to detect Alzheimer's Disease from MRI brain images. A Convolutional Neural Network (CNN) is trained to classify MRI scans into different dementia categories.

## Features
- MRI image preprocessing
- Deep Learning model for classification
- High accuracy prediction
- Jupyter Notebook implementation
- Confusion Matrix evaluation

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Pandas
- Matplotlib
- OpenCV
- Scikit-Learn

## Dataset
MRI brain image dataset for Alzheimer's Disease detection.

## How to Run

1. Install required libraries.
2. Open the notebook.
3. Run all cells.
4. Upload MRI images for prediction.

## Results

- Training Accuracy: 99.99%
- Test Accuracy: 99.94%
- Model: Convolutional Neural Network (CNN)
- Dataset: MRI Brain Images
- Task: Alzheimer's Disease Classification
